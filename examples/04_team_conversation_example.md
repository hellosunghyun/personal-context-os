# 팀 대화 예시

[← 이전](03_final_manual_excerpt.md) | [목차](../SUMMARY.md) | [다음 →](../tests/manual_quality_test.md)

## 상황

두 사람이 같은 문제를 두고 계속 의견이 엇갈립니다.

## GPT에 묻기

```text
아래는 두 사람이 공유한 협업용 사용 설명서 요약이다.
두 사람 중 누가 맞는지 판단하지 말고, 서로 어떤 맥락 차이 때문에 말이 막히는지 정리해줘.
마지막에는 직접 대화에서 사용할 확인 질문 5개만 제안해줘.
```

## 실제 대화에서 쓰는 문장

```text
제가 이해한 게 맞는지 확인하고 싶어요.
혹시 이 의견은 “속도보다 나중 유지보수 비용이 더 걱정된다”는 맥락에서 나온 걸까요?
```

## 주의

상대의 설명서를 근거로 상대를 단정하지 않습니다. 설명서는 직접 대화를 위한 질문을 만드는 보조 도구입니다.
