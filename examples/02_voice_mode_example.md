# 음성 진행 예시

[← 이전](01_text_mode_example.md) | [목차](../SUMMARY.md) | [다음 →](03_final_manual_excerpt.md)

## 상황

사용자는 글로 쓰면 생각이 너무 정제되어서, 먼저 음성으로 편하게 말하고 싶습니다.

## 진행

1. ChatGPT에 [한 문제씩 진행하는 프롬프트](../prompts/02_one_question_mode.md)를 붙여넣습니다.
2. 음성 대화를 켭니다.
3. 사용자가 말합니다.

```text
질문을 시험처럼 하지 말고 자연스러운 인터뷰처럼 해줘.
저는 말로 답할게.
제가 답변하는 동안 끊지 말고, 끝나면 짧게 정리하고 다음 질문으로 넘어가줘.
```

## 대화 느낌

```text
ChatGPT:
요즘 본인을 가장 잘 설명하는 말이 있다면 뭐라고 말할 수 있을까요?
너무 멋진 문장이 아니어도 됩니다. 그냥 요즘의 상태에 가까운 말이면 좋아요.

사용자:
음... 저는 뭔가 감은 빨리 오는데 그걸 설명하는 게 늦은 사람 같아. 그러니까 회의에서 이건 아닌데 싶은 건 바로 느끼는데, 왜 아닌지를 말하려면 과거 경험들이 다 섞여서 바로 정리가 안 되는 느낌?
```

## 끝난 뒤 반드시 할 일

```text
방금 음성 대화를 100문 100답 기록지 형식으로 정리해줘.
제가 실제로 말한 내용과 네 해석을 분리해줘.
추가 확인이 필요한 부분도 표시해줘.
```
