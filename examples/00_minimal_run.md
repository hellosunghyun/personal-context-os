# 최소 실행 예시

[목차](../SUMMARY.md) | [다음 →](01_text_mode_example.md)

## 상황

사용자는 시간이 없어서 전체 100문을 못 합니다. 먼저 10개만 답합니다.

## 진행

1. [100문 100답 기록지](../templates/answer_log.md)를 복사합니다.
2. [현재 모습 질문](../questions/01_identity.md)의 Q1~Q10에 답합니다.
3. ChatGPT에 붙여넣고 말합니다.

```text
아래는 개인 사용 설명서를 만들기 위한 초기 답변이다.
아직 10개 질문만 답했다.
이 답변을 바탕으로 제가 어떤 질문을 더 답하면 좋을지 10개만 만들어줘.
답을 추정하지 말고 질문만 만들어줘.
```

## 결과

최소한의 추가 질문을 얻고, 다음 답변으로 이어갈 수 있습니다.
