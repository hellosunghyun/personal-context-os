# 100문 100답 기록지

[← 이전](source_register.md) | [목차](../SUMMARY.md) | [다음 →](interview_summary.md)

100문 100답을 바로 작성할 수 있는 기록지입니다. 질문 원문은 아래에 모두 들어 있습니다.

질문별 설명과 예시까지 보고 싶다면 [질문 사용법](../questions/00_how_to_use_questions.md)에서 각 질문 묶음을 열어보세요.

---

## 답변 형식

```text
Q번호. 질문
- 답변:
- 실제 장면 또는 예시:
- 지금도 유효한가: 예 / 아니오 / 모르겠음
- 공개 범위: 공개 가능 / 팀원용 / 본인용만 / 비공개
- 확실도: 높음 / 중간 / 낮음
- 패스 여부: 답변 / 패스 / 나중에
```
