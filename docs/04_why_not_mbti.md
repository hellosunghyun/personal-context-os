# MBTI로 대체할 수 없는 이유

[← 이전](03_security.md) | [목차](../SUMMARY.md) | [다음 →](05_method_comparison.md)

## 여기서 볼 것

MBTI, 성향검사, 커리어 진단이 왜 유용하지만 충분하지는 않은지 설명합니다.

---

## MBTI는 쓸모없다는 뜻이 아닙니다

MBTI나 각종 성향검사는 좋은 시작점이 될 수 있습니다.

```text
선호하는 방식
타인과 다른 경향
대화의 출발점
자기이해의 언어
```

하지만 이 가이드는 유형을 찾는 방식이 아닙니다.

출처 메모: MBTI를 선호 경향의 언어로 보는 기본 설명은 [Myers-Briggs 공식 개요](../references.md#심리와-성격-개념)를 참고했습니다. 다만 이 문서는 MBTI의 타당도 논쟁을 다루려는 글이 아니므로, 성격검사는 참고 자료로만 씁니다.

---

## 왜 부족한가

사람이 실제로 만족하는 답변은 유형보다 맥락에 더 크게 좌우됩니다.

예를 들어 같은 INTJ라도 다를 수 있습니다.

```text
어떤 사람은 짧은 결론을 좋아한다.
어떤 사람은 긴 근거가 있어야 납득한다.
어떤 사람은 리스크를 먼저 확인해야 움직인다.
어떤 사람은 말투가 자기 말 같지 않으면 쓰지 못한다.
어떤 사람은 반대 관점까지 있어야 신뢰한다.
```

이 차이는 네 글자 유형보다 실제 답변과 사례에서 더 잘 드러납니다.

출처 메모: 성격을 더 연속적인 특성으로 다루는 배경은 Big Five 문헌에서도 볼 수 있습니다. 이 가이드는 MBTI와 Big Five를 대체 평가 도구로 쓰기보다, 실제 사례와 답변 기록을 모으는 출발점으로만 다룹니다. 관련 문헌은 [심리와 성격 개념](../references.md#심리와-성격-개념)에 모았습니다.

---

## 이 가이드가 수집하는 것

```text
실제 장면
반복된 선택
불편했던 피드백
납득했던 설명
싫었던 말투
좋았던 답변 구조
처음 상상했던 답변과 실제 출력의 차이
```

요즘 LLM은 이런 디테일한 맥락과 뉘앙스를 꽤 잘 읽습니다. 그래서 검사 결과보다 답변 기록과 사례가 더 강력한 참고자료가 될 때가 많습니다.

---

## 검사 결과를 쓰고 싶다면

검사 결과는 정답이 아니라 질문 생성 재료로 쓰세요.

```text
검사 결과: 구조화된 환경을 선호함
추가 질문: 실제로 어떤 구조가 있으면 편한가?
추가 질문: 반대로 어떤 구조는 통제처럼 느껴지는가?
추가 질문: 최근 1년 동안 이 성향이 드러난 장면은?
```
