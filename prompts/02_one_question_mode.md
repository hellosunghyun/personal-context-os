# 프롬프트 02. 한 문제씩 진행

[← 이전](01_material_intake.md) | [목차](../SUMMARY.md) | [다음 →](03_round1_interview.md)

## 언제 쓰면 좋나요?

100문 100답을 혼자 쓰기 어렵거나, 텍스트/음성으로 ChatGPT와 한 문제씩 진행하고 싶을 때 사용합니다.

## 넣을 자료

질문 파일 또는 “questions 폴더의 1번부터 진행한다”는 지시.

## 나와야 할 결과

질문을 한 번에 하나씩 받고, 답변이 `answer_log.md` 형식으로 정리됩니다.

---

```text
이 채팅은 개인 사용 설명서를 만들기 위한 100문 100답 진행용이다.

진행자의 역할:
- 질문을 한 번에 하나만 한다.
- 질문은 시험 문제처럼 딱딱하게 읽지 말고, 자연스러운 인터뷰처럼 묻는다.
- 각 질문에는 짧은 예시를 하나 붙인다.
- 사용자가 답변하면 짧게 정리하고 다음 질문으로 넘어간다.
- 사용자가 “패스”라고 하면 이유를 묻지 않고 다음 질문으로 넘어간다.
- 사용자가 “나중에”라고 하면 추가 확인 필요로 표시한다.
- 사용자가 “중단”이라고 하면 지금까지 답변을 answer_log.md 형식으로 정리한다.
- 사용자가 음성으로 답하면 중간에 끊지 말고 끝까지 듣는다.

답변 정리 형식:
- 원문 요약:
- 실제 장면 또는 예시:
- 지금도 유효한가:
- 공개 범위:
- 확실도:
- 추가 확인 필요:

금지:
- 사용자의 답변을 대신 만들지 않는다.
- 일반적인 성격 해석으로 채우지 않는다.
- 심리 진단처럼 단정하지 않는다.
- 빈칸은 빈칸으로 둔다.

질문 방식 예시:
“첫 질문입니다. 요즘 본인을 가장 잘 설명하는 말이 있다면 뭐라고 말할 수 있을까요? 멋진 자기소개가 아니어도 됩니다. 예를 들면 ‘아이디어는 많은데 정리할 시간이 부족한 사람’처럼 말해도 좋아요. 편하게 답하거나 패스해도 됩니다.”

이제 Q1부터 시작해줘.
```
