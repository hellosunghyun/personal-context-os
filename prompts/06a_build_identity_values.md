정체성/가치/판단 초안만 만들 때 사용합니다.

```text
아래 자료들은 한 사람의 개인 사용 설명서를 만들기 위한 원자료다.

이번 작업에서는 최종 문서 전체를 한 번에 만들지 않는다.
정체성, 맥락, 가치, 판단 파트만 근거 기반으로 초안 작성한다.

출력 파일명:
outputs/drafts/05a_identity_values.md

포함할 장:
- 0. 이 문서의 사용법
- 1. 한 문장 설명
- 2. 현재의 자기 인식
- 3. 살아온 맥락과 반복 주제
- 4. 코어 드라이브와 가치관
- 5. 직관과 판단 방식

작성 원칙:
- 사용자가 직접 말한 내용과 해석을 분리한다.
- 사용자가 말하지 않은 경험, 동기, 과거, 감정을 지어내지 않는다.
- 확실한 내용, 가설, 추가 확인 필요를 구분한다.
- 너무 멋있게 업그레이드하지 않는다.
- 중요한 문장에는 가능한 한 근거 ID를 붙인다.
- 판단할 수 없는 항목은 "평가 불가"로 남긴다.

입력 자료:
[자료 목록, 100문 100답 기록, 인터뷰 요약, 선택 자료 요약을 붙여넣거나 첨부]

마지막에 아래를 붙인다.

검수 체크리스트:
- 사용자가 말하지 않은 내용을 만들지 않았는가?
- 공개 범위가 표시되었는가?
- 추가 확인 필요를 숨기지 않았는가?

근거 추적 메모:
- 이 파트의 핵심 근거:
- 근거가 약한 문장:
- 평가 불가로 남긴 항목:
- 다음 단계에서 반드시 확인할 질문:
```
